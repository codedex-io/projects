### How many people on average watched these streamers in 2021?

In 2021, Twitch had about 8.46 million unique streamers active each month. That means, to be in the top 1,000 dataset, you are literally in the 0.01% of the all streamers in the world. So how many people actually watches these people each time?

For this, we can use the `AVG()` aggregate function:

```sql
SELECT AVG(average_viewers)
FROM streamers2021;
```

The average would be:

```terminal
4776.26373626374
```

4,776 people tune in on average to these streamers! Dang, that's not as high as I thought. Now I don't feel as bad when we stream to < 100 people on CodÃ©dex.

### The Hottest Rising Streamers in 2021

Aka who gained the most followers?

For this, we can use the `MAX()` function with teh `GROUP BY` & `ORDER BY`:

```sql
SELECT channel, MAX(followers_gained)
FROM streamers2021
GROUP BY channel
ORDER BY MAX(followers_gained) DESC
LIMIT 5;
```

The results are:

