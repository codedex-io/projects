### How many people on average watched these streamers in 2021?

In 2021, Twitch had about 8.46 million unique streamers active each month. That means, to be in the top 1,000 dataset, you are literally in the 0.01% of the all streamers in the world. So how many people actually watches these people each time?

For this, we can use the `AVG()` aggregate function:

```sql
SELECT AVG(average_viewers)
FROM streamers2021;
```

The average would be:

```plaintext
4776.26373626374
```

4,776 people tune in on average to these streamers! Dang, that's not as high as I thought. Now I don't feel as bad when we stream to < 100 people on [CodÃ©dex's Twitch](https://www.twitch.tv/codedex_io).

### The Hottest Rising Streamers in 2021

Aka who gained the most followers?

For this, we can use the `MAX()` function with the `GROUP BY` & `ORDER BY`:


## Grouping Top Games Into Genres in 2024

So earlier, we found the most popular games in 2024. Let's do it with a `LIMIT 25` this time:

```sql
SELECT most_streamed_game, COUNT(*)
FROM streamers2024
GROUP BY 1
ORDER BY 2 DESC
LIMIT 30;
```

With "Just Chatting" ended up so high up, I wonder if we group some of the top games, that it would come close.

Let's try to figure out the different genres of games, and each of its viewer counts.

Group the games into their genres: 

- [IRL](https://en.wikipedia.org/wiki/Real_life)
- [MOBA: Multiplayer Online Battle Arena](https://en.wikipedia.org/wiki/Multiplayer_online_battle_arena)
- [FPS: First-Person Shooter](https://en.wikipedia.org/wiki/First-person_shooter)
- [Battle Royale](https://en.wikipedia.org/wiki/Battle_royale_game)
- MMO: [Massively Multiplayer Online](https://en.wikipedia.org/wiki/Massively_multiplayer_online_game)
- Simulation
- Action
- Gambling

This can be done using a `CASE` statement, which is similar to an `if`/`else if`/`else` or `switch` statement in Python, C++, or Java.

Using `CASE`, your logic should be: 

  - If League of Legends or Dota 2 â†’ MOBA 
  - If Valorant or Counter-Strike or Overwatch or Escape from Tarkov â†’ FPS 
  - If Minecraft â†’ Simulation
  - etc. etc.
  - Else â†’ Other 

Use `GROUP BY` and `ORDER BY` to showcase only the unique game titles.

```sql
SELECT
  CASE
    WHEN most_streamed_game = 'Just Chatting'
      THEN 'IRL'
    WHEN most_streamed_game = 'League of Legends' OR most_streamed_game = 'Dota 2'
      THEN 'MOBA'
    WHEN most_streamed_game = 'VALORANT' OR most_streamed_game = 'Valorant' OR most_streamed_game = 'Counter-Strike' OR most_streamed_game = 'Overwatch' OR most_streamed_game = 'Escape from Tarkov'
      THEN 'FPS'
    WHEN most_streamed_game = 'Minecraft' or most_streamed_game = 'Dead by Daylight'
      THEN 'Simulation'
    WHEN most_streamed_game = 'Fortnite' OR most_streamed_game = 'Apex Legends' OR most_streamed_game = 'Call of Duty: Warzone' OR most_streamed_game = 'PUBG: BATTLEGROUNDS'
      THEN 'Battle Royale'
    WHEN most_streamed_game = 'World of Warcraft'
      THEN 'MMO'
    WHEN most_streamed_game = 'Casino' OR most_streamed_game = 'Virtual Casino' OR most_streamed_game = 'Slots'
      THEN 'Gambling'
    WHEN most_streamed_game = 'Sports' OR most_streamed_game = 'FIFA 23' OR most_streamed_game = 'Rocket League'
      THEN 'Sports'
    WHEN most_streamed_game = 'Hearthstone' OR most_streamed_game = 'Teamfight Tactics' or most_streamed_game = 'Chess'
      THEN 'Strategy'
  ELSE 'Other'
  END AS 'genre',
  COUNT(*)
FROM streamers2024
GROUP BY 1
ORDER BY 2 DESC;
```

The result should look something like:

```plaintext
IRL|257
Other|234
FPS|123
MOBA|119
Battle Royale|88
Gambling|68
Simulation|33
Sports|30
Strategy|29
MMO|18
```

Wild to see "Just Chatting" still so ahead!
