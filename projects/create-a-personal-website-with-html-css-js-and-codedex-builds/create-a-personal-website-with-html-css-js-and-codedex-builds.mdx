## Introduction

If you didn't already know, having a Codédex account gives you access to one of the biggest features on the site: ✨Builds✨, an online code editor where you can run code in different languages, as well as creating a website! 

And recently, we launched Builds 1.0! 🚀

In Codédex Builds 1.0, a new opportunity opens up for you to not only code and see your output, but actually deploy your websites with a link to share! You can also:

- 📄 **Add new files:** So you can have multiple pages and styles on your website.
- 🖼️ **Add assets:** Yep! You can host images, GIFs, and videos right on Codédex.
- 🔗 **Create a custom share link:** Grab yours today!

In this project tutorial, we're going to show you how to create your custom HTML/CSS/JS website with Codédex Builds, and share some templates to get started! ⭐

## Set Up

Navigate to the [Build](https://www.codedex.io/builds) page on your account. 

<ImageZoom src="https://firebasestorage.googleapis.com/v0/b/codedex-io.appspot.com/o/projects%2Fcreate-a-personal-website-with-html-css-js-and-codedex-builds%2F001.webp?alt=media&token=14376266-41f2-49d5-86b0-74191ecf7b01" alt="New builds" style={{ width: "80%", height: "auto" }} />

From here, click on “+ New Build”, and select the “HTML/CSS/JS” option! 

<ImageZoom src="https://firebasestorage.googleapis.com/v0/b/codedex-io.appspot.com/o/projects%2Fcreate-a-personal-website-with-html-css-js-and-codedex-builds%2F002.webp?alt=media&token=59ef536d-99e7-4c80-9058-82c86e27b307" alt="Select HTML/CSS/JS" style={{ width: "80%", height: "auto" }} />

You'll be redirected to the editing page! This is where you'll be able to code your website. Now's a good time to give your website a title.

<ImageZoom src="https://firebasestorage.googleapis.com/v0/b/codedex-io.appspot.com/o/projects%2Fcreate-a-personal-website-with-html-css-js-and-codedex-builds%2F003.webp?alt=media&token=016852eb-0b42-408d-8469-b068e0b4d057" alt="Editing page" style={{ width: "80%", height: "auto" }} />

Here, you're going to find the HTML, CSS, and JavaScript files to get started on your project. 

## Websites

From here, you're welcome to get started on customizing your very own website that you intend on sharing. Some popular options include: 

- The classic About Me page: a landing portfolio page that shares about you, how to contact you, and a picture! 
- The newsletter page: Host a blog directly on your website, and add all the articles you want! 
- The linktree dupe: Want a quick links page to share projects or contact info? This option is for you! 

Take a look at what all three look like, pick a template, and get started! ✨

### Links Page

<ImageZoom src="https://firebasestorage.googleapis.com/v0/b/codedex-io.appspot.com/o/projects%2Fcreate-a-personal-website-with-html-css-js-and-codedex-builds%2Fquick-links-page.webp?alt=media&token=6f223e00-f77c-437a-83db-371adf148c08" alt="Links page" style={{ width: "80%", height: "auto" }} />

A one-stop-shop for all your links, and a small description about yourself! Plug your socials here too! 

This one won't require another page, but you can add your own photo with the “Assets” feature! 

Click on “File +”, and click “Upload asset”. 

<ImageZoom src="https://firebasestorage.googleapis.com/v0/b/codedex-io.appspot.com/o/projects%2Fcreate-a-personal-website-with-html-css-js-and-codedex-builds%2F005.webp?alt=media&token=b3a5b89b-5173-4722-b998-6323c2242de5" alt="Upload asset" style={{ width: "80%", height: "auto" }} />

Next, select any file from your computer. 

<ImageZoom src="https://firebasestorage.googleapis.com/v0/b/codedex-io.appspot.com/o/projects%2Fcreate-a-personal-website-with-html-css-js-and-codedex-builds%2F006.webp?alt=media&token=7c0825c1-71dd-4f0f-9f04-bb62dba46374" alt="Upload asset" style={{ width: "80%", height: "auto" }} />

Then, you'll see your photo ready to use! “Copy relative path” will let you use the link to access it when you link it as a source to an image. 

```html
<img src=”file-path-here” alt=”” /> 
```
