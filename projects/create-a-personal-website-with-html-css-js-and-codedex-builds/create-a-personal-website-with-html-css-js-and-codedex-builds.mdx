## Set Up

Navigate to the [Build](https://www.codedex.io/builds) page on your account. 

<ImageZoom src="https://firebasestorage.googleapis.com/v0/b/codedex-io.appspot.com/o/curriculum%2Fimages%2Fproject-codedex-builds%2F001.png?alt=media&token=f82c0da4-2ceb-43b5-b873-6ff9aa649967" alt="New builds" style={{ width: "100%", height: "auto" }} />

From here, click on “+ New Build”, and select the “HTML/CSS/JS” option! 

<ImageZoom src="https://firebasestorage.googleapis.com/v0/b/codedex-io.appspot.com/o/curriculum%2Fimages%2Fproject-codedex-builds%2F002.png?alt=media&token=b1133e8a-9fda-445b-91b5-101309558da5" alt="Select HTML/CSS/JS" style={{ width: "100%", height: "auto" }} />

You'll be redirected to the editing page! This is where you'll be able to code your website. Now's a good time to give your website a title.

<ImageZoom src="https://firebasestorage.googleapis.com/v0/b/codedex-io.appspot.com/o/curriculum%2Fimages%2Fproject-codedex-builds%2F003.png?alt=media&token=46d87ba7-37c8-4f70-896f-e706dce0df17" alt="Editing page" style={{ width: "100%", height: "auto" }} />

### 1. About Me

<ImageZoom src="https://firebasestorage.googleapis.com/v0/b/codedex-io.appspot.com/o/curriculum%2Fimages%2Fproject-codedex-builds%2Fabout-me-page.png?alt=media&token=a1a36772-b44f-44fb-89ab-4c6cb9fa1954" alt="About Me page" style={{ width: "100%", height: "auto" }} />

The About Me homepage is inspired by a 90s retro look with an about me, interests section, and fun links! 

Take a look at
- [The Demo](about.codedex.me) 
- [The Code Template](https://www.codedex.io/@ellie/build/template-about-me-home)

One bonus for this template is the concept of a visitor counter! This is a fun way to see how many people have visited your website. 

The JavaScript code is in the template file, so you can easily add it to your own website.

```js
document.addEventListener('DOMContentLoaded', () => {
  const countEl = document.getElementById('visitor-count');
  const dateEl = document.getElementById('last-updated');

  if (countEl) {
    const key = 'about-me-visitor-count';
    let count = parseInt(localStorage.getItem(key) || '0', 10) + 1;
    localStorage.setItem(key, count);
    countEl.textContent = String(count).padStart(6, '0');
  }
```

### 2. Newsletter / Blog

<ImageZoom src="https://firebasestorage.googleapis.com/v0/b/codedex-io.appspot.com/o/curriculum%2Fimages%2Fproject-codedex-builds%2Fnewsletter-page.png?alt=media&token=a2ecb114-903e-4516-91b1-4852118b2e1d" alt="Newsletter page" style={{ width: "100%", height: "auto" }} />

This page is a take on [Substack](https://substack.com/) or [Medium](https://medium.com/). You can create a site that lets you make articles and blogs that you can share! 

<ImageZoom src="https://firebasestorage.googleapis.com/v0/b/codedex-io.appspot.com/o/curriculum%2Fimages%2Fproject-codedex-builds%2Fnewsletter-page-2.png?alt=media&token=ef202cd0-38b7-47a2-90dc-80e64ea9dcc2" alt="Newsletter page" style={{ width: "100%", height: "auto" }} />

The website will require you to use a new feature, the “Create File” button. Go to “File +” and click the button to create a new page called **article.html**. 

<ImageZoom src="https://firebasestorage.googleapis.com/v0/b/codedex-io.appspot.com/o/curriculum%2Fimages%2Fproject-codedex-builds%2F004.png?alt=media&token=6bea6866-5476-44e3-a426-ba0cbc7409e3" alt="Newsletter page" style={{ width: "100%", height: "auto" }} />

You will see a template for an article page, which will hold an article that you write! 

```html
<article class="article">
  <header class="article-header">
    <h1>Why Slow Mornings Changed How I Write</h1>
    <p class="article-subtitle">
      I used to open my laptop before my eyes were fully open. Here's what
      happened when I stopped.
    </p>

    <div class="article-byline">
      <img
        class="author-avatar author-avatar--small"
        src="https://api.dicebear.com/7.x/notionists/svg?seed=Jordan"
        alt="Jordan Lee"
        width="48"
        height="48"
      />
      <div>
        <span class="author-name">Jordan Lee</span>
        <span class="article-meta">
          <time datetime="2026-06-03">Jun 3, 2026</time>
          · 6 min read
        </span>
      </div>
    </div>
  </header>

  <div class="article-body">
    <p>
      For years, my morning routine looked like this: alarm, phone, email,
      Slack, coffee somewhere in the middle if I remembered. Writing happened
      whenever I could squeeze it in — usually late at night, usually tired.
    </p>

  <footer class="article-footer">
    <p class="article-thanks">Thanks for reading The Weekly Note.</p>
    <a href="index.html" class="back-link">← Back to all posts</a>

    <form class="subscribe-box subscribe-box--inline" id="subscribe-form">
      <label for="email">Enjoyed this? Subscribe for free</label>
      <div class="subscribe-row">
        <input
          type="email"
          id="email"
          name="email"
          placeholder="your@email.com"
          required
        />
        <button type="submit">Subscribe</button>
      </div>
      <p class="subscribe-note" id="subscribe-message">
        New posts every Tuesday. No spam.
      </p>
    </form>
  </footer>
</article>
```

This will hold an article that you write! Create more files for future articles. 

Take a look at
- [The Demo](newsletter.codedex.me)   
- [The Code Template](https://www.codedex.io/@ellie/build/template-newsletter-blog) 


### 3. Links Page

<ImageZoom src="https://firebasestorage.googleapis.com/v0/b/codedex-io.appspot.com/o/curriculum%2Fimages%2Fproject-codedex-builds%2Fquick-links-page.png?alt=media&token=eafcaa36-afce-4fc6-b00b-410afad8506a" alt="Links page" style={{ width: "100%", height: "auto" }} />

A one-stop-shop for all your links, and a small description about yourself! Plug your socials here too! 

This one won't require another page, but you can add your own photo with the “Assets” feature! 

Click on “File +”, and click “Upload asset”. 

<ImageZoom src="https://firebasestorage.googleapis.com/v0/b/codedex-io.appspot.com/o/curriculum%2Fimages%2Fproject-codedex-builds%2F005.png?alt=media&token=5df0232d-c167-4267-9b30-3351fb157a1d" alt="Upload asset" style={{ width: "100%", height: "auto" }} />

Next, select any file from your computer. 

<ImageZoom src="https://firebasestorage.googleapis.com/v0/b/codedex-io.appspot.com/o/curriculum%2Fimages%2Fproject-codedex-builds%2F006.png?alt=media&token=a129b523-2a58-4764-aeba-3fa34ab9d836" alt="Upload asset" style={{ width: "100%", height: "auto" }} />

Then, you'll see your photo ready to use! “Copy relative path” will let you use the link to access it when you link it as a source to an image. 

```html
<img src=”file-path-here” alt=”” /> 
```

Once you've uploaded your photo, you can use it in your website!

Then, you can create links to your other pages, and add a description for each link.

```html
<!-- Link to a page -->
<a class="link-btn" href="#" target="_blank" rel="noopener noreferrer">
  <span class="link-icon" aria-hidden="true">🛍️</span>
  <span class="link-text">Shop Prints &amp; Stickers</span>
  <span class="link-arrow" aria-hidden="true">→</span>
</a>
```
