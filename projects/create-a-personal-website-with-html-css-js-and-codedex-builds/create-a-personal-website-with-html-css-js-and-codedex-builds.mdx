## Introduction

If you didn't already know, having a Codédex account gives you access to one of the biggest features on the site: ✨Builds✨, an online code editor where you can run code in different languages, as well as creating a website! 

And recently, we launched Builds 2.0! 🚀

<ImageZoom src="https://raw.githubusercontent.com/codedex-io/projects/main/projects/create-a-personal-website-with-html-css-js-and-codedex-builds/images/builds-feature-demo.gif" alt="Codédex Builds 2.0" style={{ width: "80%", height: "auto" }} />

In Codédex Builds 2.0, a new opportunity opens up for you to not only code and see your output, but actually deploy your websites with a link to share! You can also:

- 📄 **Add new files:** So you can have multiple pages and styles on your website.
- 🖼️ **Add assets:** Yep! You can host images, GIFs, and videos right on Codédex.
- 🔗 **Create a custom share link:** Grab yours today!

In this project tutorial, we're going to show you how to create your custom HTML/CSS/JS website with Codédex Builds, and share some templates to get started! ⭐

Here's some of the pages we're going to make today: 
- [About Me webiste]
- [Newsletter / Blog]
- [Quick Links Page]


<ImageZoom src="https://raw.githubusercontent.com/codedex-io/projects/main/projects/create-a-personal-website-with-html-css-js-and-codedex-builds/images/profile_examples.gif" alt="Website pages" style={{ width: "80%", height: "auto" }} />

Let's get started!
## Set Up

Navigate to the [Build](https://www.codedex.io/builds) page on your account. 

<ImageZoom src="https://raw.githubusercontent.com/codedex-io/projects/main/projects/create-a-personal-website-with-html-css-js-and-codedex-builds/images/001.png" alt="New builds" style={{ width: "80%", height: "auto" }} />

From here, click on “+ New Build”, and select the “HTML/CSS/JS” option! 

<ImageZoom src="https://raw.githubusercontent.com/codedex-io/projects/main/projects/create-a-personal-website-with-html-css-js-and-codedex-builds/images/002.png" alt="Select HTML/CSS/JS" style={{ width: "80%", height: "auto" }} />

You'll be redirected to the editing page! This is where you'll be able to code your website. Now's a good time to give your website a title.

<ImageZoom src="https://raw.githubusercontent.com/codedex-io/projects/main/projects/create-a-personal-website-with-html-css-js-and-codedex-builds/images/003.png" alt="Editing page" style={{ width: "80%", height: "auto" }} />

Here, you're going to find the HTML, CSS, and JavaScript files to get started on your project. 

## Websites

From here, you're welcome to get started on customizing your very own website that you intend on sharing. Some popular options include: 

- The classic About Me page: a landing portfolio page that shares about you, how to contact you, and a picture! 
- The newsletter page: Host a blog directly on your website, and add all the articles you want! 
- The linktree dupe: Want a quick links page to share projects or contact info? This option is for you! 

Take a look at what all three look like, pick a template, and get started! ✨

### Links Page

<ImageZoom src="https://raw.githubusercontent.com/codedex-io/projects/main/projects/create-a-personal-website-with-html-css-js-and-codedex-builds/images/quick-links-page.png" alt="Links page" style={{ width: "80%", height: "auto" }} />

A one-stop-shop for all your links, and a small description about yourself! Plug your socials here too! 

This one won't require another page, but you can add your own photo with the “Assets” feature! 

Click on “File +”, and click “Upload asset”. 

<ImageZoom src="https://raw.githubusercontent.com/codedex-io/projects/main/projects/create-a-personal-website-with-html-css-js-and-codedex-builds/images/005.png" alt="Upload asset" style={{ width: "80%", height: "auto" }} />

Next, select any file from your computer. 

<ImageZoom src="https://raw.githubusercontent.com/codedex-io/projects/main/projects/create-a-personal-website-with-html-css-js-and-codedex-builds/images/006.png" alt="Upload asset" style={{ width: "80%", height: "auto" }} />

Then, you'll see your photo ready to use! “Copy relative path” will let you use the link to access it when you link it as a source to an image. 

```html
<img src=”file-path-here” alt=”” /> 
```
